# Prompt Engineering Essentials - Course Guide

## Course Context
This is a mini course for writingIO focused on writing effective AI prompts.

## Video Script Guidelines
- Target length: 1-2 minutes per video (roughly 900 characters per minute)
- Use "A I" (with space) for proper text-to-speech pronunciation
- Include practical examples when possible
- Focus on immediately applicable techniques

## Lesson Topics
1. Why Prompts Matter - Importance of prompt quality
2. Anatomy of a Good Prompt - Key prompt components
3. Context and Specificity - Adding useful detail
4. Iterating and Refining - Improving through iteration
5. Templates You Can Use - Practical prompt templates
