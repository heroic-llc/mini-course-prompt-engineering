# Prompt Engineering Essentials

A mini course on writing effective prompts to get better results from AI.

## Course Overview

This course teaches why prompts matter, the anatomy of a good prompt, how to add context and specificity, techniques for iterating and refining, and provides reusable templates.

## Lessons

1. **Why Prompts Matter** - The importance of good prompts
2. **Anatomy of a Good Prompt** - Key components of effective prompts
3. **Context and Specificity** - Adding detail for better results
4. **Iterating and Refining** - Improving prompts through iteration
5. **Templates You Can Use** - Ready-to-use prompt templates
